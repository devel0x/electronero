Electronero

Roadmap

2018 – Project Inception

Forked from Monero and Electroneum with additional security updates.

2020 – RPC Libraries

Node.js, PHP, Go and Python modules added for easy RPC integration.

2021 – EI-2.0 Smart Chain

Launch with cross-chain atomic swaps and xAssets support.

2021 – XRC-20 Token Platform

Lightweight token system accessible via CLI and RPC.

Future

Governance features and extended token utilities.
